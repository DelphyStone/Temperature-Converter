# Temperature Converter
This is a basic temperature converter website that allows users to convert temperatures between Celsius and Fahrenheit.

## Features
* Convert Celsius to Fahrenheit
* Convert Fahrenheit to Celsius

## Technologies Used
**HTML:** Used to create the structure of the website.
**CSS:** Used for styling and layout.
**JavaScript:** Used to make the website functional and perform the temperature conversions.
